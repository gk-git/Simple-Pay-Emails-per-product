== Description ==

Stipe Simple Pay Different Email Per Product is an extension of the Stripe Simple Pay plugin that allows having a different customer notification email for each product

For the plugin to work correctly you would need to have:
 * [ACF](https://wordpress.org/plugins/advanced-custom-fields/) installed and activated
 * [Stripe Payments WordPress Plugin – WP Simple Pay](https://wordpress.org/plugins/stripe/) installed and activated


== Installation ==

The easiest way to install Stipe Simple Pay Different Email Per Product is to search for it via your WordPress Dashboard.

= Step-by-step instructions =

1. Log in to your *WordPress* dashboard (e.g. www.yourdomain.com/wp-admin).
2. Click on the *Plugins* tab in the left panel, then click *Add New*.
3. Search for *Stipe Simple Pay Different Email Per Product* and find our plugin near the top.
4. Install it by clicking the *Install Now* link.
5. When installation finishes, click *Activate Plugin*.
6. A new menu item *Stipe Simple Pay Different Email Per Product* should appear in your dashboard.
